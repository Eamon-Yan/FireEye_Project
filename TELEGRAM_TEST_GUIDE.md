# Telegram 测试指南

## 快速测试步骤

### 1. 打开 Telegram
找到你的 AstrBot 机器人

### 2. 测试帮助命令
发送以下任一命令：
```
/火瞳帮助
```
或
```
/ht帮助
```

**预期结果**: 显示帮助信息，列出所有可用命令

---

### 3. 测试查询命令
发送以下任一命令：
```
/火瞳查询 火灾
```
或
```
/ht查询 火灾事故
```

**预期结果**: 
- 显示查询结果
- 列出相关节点（事件、实体等）
- 显示会话 ID

---

### 4. 测试统计命令
发送以下任一命令：
```
/火瞳统计
```
或
```
/ht统计
```

**预期结果**:
- 显示系统统计信息
- 事件总数: 26
- 实体总数: 26
- 关系总数: 17
- 节点类型和关系类型分布

---

## 故障排查

### 如果命令没有响应

1. **检查插件是否加载**
   ```bash
   docker logs astrbot 2>&1 | grep -i "fire-eye"
   ```
   应该看到: `[Fire-Eye] Plugin initialized`

2. **检查后端是否运行**
   ```bash
   docker ps | grep fire-eye-backend
   ```
   应该看到容器正在运行

3. **测试后端 API**
   ```bash
   docker exec astrbot curl -s http://fire-eye-backend:8000/health
   ```
   应该返回: `{"status":"healthy"}`

4. **重启 AstrBot**
   ```bash
   docker restart astrbot
   ```
