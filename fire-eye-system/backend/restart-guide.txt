问题已修复！但需要重启后端服务才能生效。

┌──────────────────────────────────────────────────────────────┐
│ 第1步：停止后端服务                                           │
└──────────────────────────────────────────────────────────────┘

  1. 找到运行后端的命令行窗口
     （标题可能是：uvicorn 或 start-backend.bat）
  
  2. 在该窗口按：Ctrl + C
  
  3. 等待服务完全停止

┌──────────────────────────────────────────────────────────────┐
│ 第2步：重新启动后端                                           │
└──────────────────────────────────────────────────────────────┘

  在同一个命令行窗口运行：
  
  start-backend.bat
  
  等待看到以下消息：
  ✓ Application startup complete
  ✓ Uvicorn running on http://0.0.0.0:8000

┌──────────────────────────────────────────────────────────────┐
│ 第3步：测试API                                                │
└──────────────────────────────────────────────────────────────┘

  打开新的命令行窗口，运行：
  
  cd fire-eye-system\backend
  test-api.bat
  
  应该看到：
  ✅ 测试 3: 查询图数据 - 成功

┌──────────────────────────────────────────────────────────────┐
│ 第4步：刷新前端                                               │
└──────────────────────────────────────────────────────────────┘

  1. 打开浏览器访问：http://localhost:8080/graph
  
  2. 按 Ctrl + Shift + R 强制刷新
  
  3. 应该能看到彩色的图谱节点：
     🔴 红色 - 火灾事件
     🟠 橙色 - 安全隐患
     🟣 紫色 - 后果影响
