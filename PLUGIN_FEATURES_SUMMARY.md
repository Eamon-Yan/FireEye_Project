**示例**: `/火瞳查询 火灾`

**状态**: 完全实现并测试通过

### 4. ✅ 文件上传命令
**命令**: `/火瞳上传` 或 `/ht上传`

- 自动下载文件到临时目录
- 验证文件类型和大小（最大10MB）
- 上传到 Fire-Eye 后端
- 启动异步分析任务
- 实时显示分析进度
- 返回提取的事件链
- 自动清理临时文件

**示例**:

## 技术架构

### 当前架构
```
用户 (Telegram/QQ/等)
    ↓
AstrBot 平台
    ↓
火瞳插件 (main.py)
    ├── 命令处理器
    │   ├── help_handler (帮助)
    │   ├── query_handler (查询)
    │   └── stats_handler (统计)
    ├── API 客户端
    │   └── client.py (HTTP 通信)
    └── 配置管理
        └── config.yaml
    ↓
Fire-Eye 后端 API
    ↓
Neo4j 知识图谱
```
