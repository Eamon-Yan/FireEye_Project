# Telegram 文件上传使用指南

## 问题说明

当你在 Telegram 中只发送 `/火瞳上传` 命令而没有附带文件时，会出现错误：
```
❌ 上传失败: 'TelegramPlatformEvent' object has no attribute 'message'
```

这是因为命令需要配合文件一起使用。

## 正确使用方法

### 方法 1：先发送文件，再发送命令（推荐）

1. 在 Telegram 中点击附件按钮 📎
2. 选择要上传的文件（支持 PDF、TXT、DOCX）
3. 在文件的说明栏中输入：`/火瞳上传`
4. 点击发送

### 方法 2：直接发送文件（最简单）

1. 在 Telegram 中点击附件按钮 📎
2. 选择要上传的文件
3. 直接发送文件（不需要任何命令）
4. 系统会自动识别并处理文件

### 方法 3：使用命令查看说明

1. 发送命令：`/火瞳上传`
2. 系统会返回上传说明
3. 按照说明发送文件

## 支持的文件格式

- PDF 文件 (.pdf)
- 文本文件 (.txt)
- Word 文档 (.docx)

## 文件大小限制

- 最大文件大小：10MB

## 常见问题

### Q: 为什么只发送命令会报错？
A: `/火瞳上传` 命令需要配合文件一起使用。如果只发送命令，系统会返回使用说明。

### Q: 可以一次上传多个文件吗？
A: 目前只支持单个文件上传。如需上传多个文件，请分别上传。

### Q: 上传后多久能看到结果？
A: 通常需要 30-60 秒，具体取决于文件大小和内容复杂度。

### Q: 上传失败怎么办？
A: 请检查：
- 文件格式是否支持（PDF/TXT/DOCX）
- 文件大小是否超过 10MB
- 网络连接是否正常
- 后端服务是否运行正常
